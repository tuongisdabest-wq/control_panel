👤 Hồ sơ hiển thị của Admin
Tên và ảnh đại diện này sẽ hiện ra ở mọi bài viết đăng trên bảng tin.

🔑 Đổi mật khẩu quản trị
Cần nhập đúng mật khẩu hiện tại. Mật khẩu mới tối thiểu 6 ký tự. Sau khi đổi, các phiên đăng nhập cũ khác vẫn còn hiệu lực tới khi hết hạn (12 giờ).

⬇️ Xuất dữ liệu sao lưu
Tải toàn bộ dữ liệu quản trị hiện tại (phim đề xuất, dòng phim, bài viết, hồ sơ) về một file JSON — dùng để lưu trữ hoặc phục hồi sau này.
⬆️ Khôi phục từ file sao lưu
Chọn 1 file JSON đã xuất trước đó. Hệ thống sẽ ghi đè dữ liệu hiện tại — hãy chắc chắn trước khi xác nhận.
🎬 Cấu hình Intro & thông tin site
Mô hình "lớp nền + override": bản gốc mặc định (intro.mp4, logo.png, tên site "KTuongFX") vẫn nằm an toàn trong code index.html. Ở đây bạn chỉ lưu phần muốn ĐÈ LÊN — để trống ô nào thì trang chủ tự dùng lại giá trị mặc định gốc của ô đó. Bấm "Khôi phục mặc định" bất cứ lúc nào để xoá sạch mọi tuỳ chỉnh, quay về đúng bản gốc.

🧊 Tạm dừng web (Freeze)
Khi bật, người xem sẽ chỉ thấy ảnh + dòng chữ bạn chọn thay vì toàn bộ trang web — dùng lúc cần sửa lỗi mà không muốn người xem thấy web bị lỗi. Các phần khác (phim, dòng phim, bài viết...) vẫn giữ nguyên, không bị mất dữ liệu.

Không chọn ảnh cũng được, chỉ cần dòng chữ bên dưới.
